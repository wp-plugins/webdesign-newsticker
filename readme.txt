== Description ==

Adds a customizeable widget which displays the latest news about webdesign. It can be integrated anywhere in the blog. This newsticker shows up the last five news. This is a very nice solution to everybody who wants to show news integrated in the blog about design.

*Feature List*

* customizable widget
* displays a user-defined number of links
* title shortening

== Installation ==

1. Extract `webdesign-newsticker.zip`.
2. Upload the `webdesign-newsticker`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets and configure the widget options.
